How to Use

Weekly routine (Monday, ~10 minutes)

  1. Open Zendesk Explore → Matt Dashboard. Set time filter to Previous week (7 days ending Sunday). Click Download to save the PDF.
  2. Open the "Matthew: Open and Pending Tickets" view in Zendesk (not "My Unsolved Tickets"). Click Actions → Export as CSV.
  3. Open the tool → Enter Week's Data.
  4. Confirm Week ending (Sunday). Enter Schedule B billing.
  5. Upload the PDF — it displays inline — type the tile numbers.
  6. Paste or upload the CSV.
  7. Glance at the portfolio scorecard. If any projects show "aged 14+d", click through to confirm or archive.
  8. Write the 3 short narrative boxes (what moved / what's stuck / asks).
  9. Preview → Save Week.

Project Register — how it works

The register is persistent — you maintain it as you work, not on Monday. Roll-ups (active / on-track / at-risk / blocked) are computed automatically from each project's current milestone.

  • Add project — pick a type (Remediation / Fire / SBBIS / Fencing / Roof / Custom). The tool pre-seeds the milestone sequence. You set the first milestone's target date; the rest cascade based on typical elapsed times.
  • Complete a milestone — tick the checkbox. The next milestone becomes current.
  • Slippage — move a target date later. The tool remembers the original and displays slippage in days.
  • On Track / At Risk / Blocked / Overdue — computed from the current milestone's target date vs today: past due = blocked, within 7 days = at risk, future = on track.
  • Aging — if you don't touch a project for 14 days, it gets flagged. The weekly form surfaces these.
  • Archive — when a project is complete, archive it. It drops off active views but stays in history.

Templates

  • Remediation (standard) — 17 milestones. Engineer → scope → tender → committee → EGM (levy + loan) → contract → works → DBP compliance → sign-off.
  • Fire / Compliance — 9 milestones. Fire order → consultant → scope → committee → (EGM if levied) → works → AFSS → certifier sign-off.
  • Defects / SBBIS — 10 milestones. Inspector → inspections → reports → developer window → rectification → reinspection → bond release.
  • Fencing (Dividing Fences Act) — 6 milestones. Notice → statutory window → agreement/NCAT → works → cost recovery.
  • Roof replacement — 10 milestones. Similar to remediation, single GM cycle.
  • Legal — defects claim — 8 milestones. Engagement → GIPA → experts → LoD → proceedings → mediation → settlement.
  • Custom — blank. Add your own milestones.

Templates are starting points. Add, remove, or rename milestones per project.
