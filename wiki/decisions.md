## [2026-04-12] META | Assumption Justification Standard adopted for all agent outputs

All agent-produced projections, forecasts, and models must now justify every assumption with: evidence, confidence level (FACT/HIGH/MEDIUM-HIGH/MEDIUM/LOW), downside risk, and invalidation signal. Scenario analysis (conservative/mid/optimistic) required for any model with 3+ assumptions. Added to CLAUDE.md as a system-wide standard.

**Why:** Agent outputs are used to allocate resources (budget, time, people). Unjustified assumptions create false confidence. Bill needs to know what's a fact, what's a strong inference, and what's speculative — and what happens if the agent is wrong.
